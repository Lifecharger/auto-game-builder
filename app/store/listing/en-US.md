# Title
Auto Game Builder

# Short Description
Remote control your dev PC from your phone — builds, AI automation, tasks.

# Full Description
Auto Game Builder is the companion app for the Auto Game Builder orchestrator. It turns your phone into a remote control panel for every game project on your dev PC.

WHAT IT DOES
• Trigger builds for Flutter, Godot, and Phaser projects with one tap
• Upload AABs directly to Google Play without opening a terminal
• Create issues (bugs, crashes, feature requests) and queue them for AI-powered auto-fix
• Track tasks per project with filters for pending, in-progress, completed, and failed
• Start and stop AI automation loops, configure intervals, and watch sessions run in real time

DOCS & KNOWLEDGE
• View and edit Game Design Documents, AI instructions (CLAUDE.md), and Art Bibles
• AI-assisted doc enhancement to tighten instructions for better agent runs

VISIBILITY
• Unified log viewer across all projects
• Build history with status badges and timestamps
• Live session monitor for ongoing automation work

AI & MCP
• Switch between Claude, Gemini, Codex, or local models per engine
• Toggle MCP servers per project: PixelLab for pixel art, ElevenLabs for audio, Meshy for 3D, device-test tools
• Context-aware chat that auto-loads the right knowledge per project type

DESIGNED FOR
Solo and small-team indie developers running multiple game projects at once who want to stay productive away from the desk.
