# Title
Auto Game Builder

# Short Description
Controla o PC de desenvolvimento do telemóvel — builds, IA, tarefas.

# Full Description
O Auto Game Builder é a app companheira do orquestrador Auto Game Builder. Transforma o teu telemóvel num painel de controlo remoto para cada projeto de jogo no teu PC de desenvolvimento.

O QUE FAZ
• Aciona builds de projetos Flutter, Godot e Phaser com um toque
• Envia AABs diretamente para o Google Play sem abrir um terminal
• Cria issues (bugs, crashes, funcionalidades) e mete-as em fila para auto-fix com IA
• Gere tarefas por projeto com filtros por pending, in-progress, completed e failed
• Inicia e para ciclos de automação de IA, configura intervalos e observa sessões ao vivo

DOCS E CONHECIMENTO
• Consulta e edita Game Design Documents, instruções de IA (CLAUDE.md) e Art Bibles
• Melhoria documental assistida por IA para afinar instruções dos agentes

VISIBILIDADE
• Visualizador unificado de logs entre projetos
• Histórico de builds com badges de estado e timestamps
• Monitor de sessão ao vivo para trabalhos de automação em curso

IA E MCP
• Alterna entre Claude, Gemini, Codex ou modelos locais por motor
• Ativa/desativa servidores MCP por projeto: PixelLab para pixel art, ElevenLabs para áudio, Meshy para 3D, ferramentas de teste
• Chat contextual que carrega o conhecimento certo conforme o tipo de projeto

PARA QUEM É
Developers indie solo e em pequenas equipas que gerem vários projetos em simultâneo e querem continuar produtivos longe da secretária.
