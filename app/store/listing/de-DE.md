# Title
Auto Game Builder

# Short Description
Steuere deinen Dev-PC vom Handy — Builds, KI-Automation, Tasks.

# Full Description
Auto Game Builder ist die Companion-App für den Auto Game Builder Orchestrator. Sie verwandelt dein Handy in ein Fernsteuerungspanel für jedes Spieleprojekt auf deinem Dev-PC.

WAS SIE TUT
• Löse Builds für Flutter-, Godot- und Phaser-Projekte mit einem Tipp aus
• Lade AABs direkt in Google Play hoch, ohne ein Terminal zu öffnen
• Erstelle Issues (Bugs, Crashes, Features) und stelle sie für KI-Auto-Fix in die Warteschlange
• Verwalte Tasks pro Projekt mit Filtern für Pending, In-Progress, Completed und Failed
• Starte und stoppe KI-Automatisierungsschleifen, stelle Intervalle ein und beobachte Sessions live

DOCS & WISSEN
• Zeige und bearbeite Game Design Documents, KI-Anweisungen (CLAUDE.md), Art Bibles
• KI-gestützte Dokumentenverbesserung für schärfere Agentenanweisungen

SICHTBARKEIT
• Einheitlicher Log-Viewer über alle Projekte hinweg
• Build-Historie mit Status-Badges und Timestamps
• Live-Session-Monitor für laufende Automatisierungen

KI & MCP
• Wechsle pro Engine zwischen Claude, Gemini, Codex oder lokalen Modellen
• Aktiviere/deaktiviere MCP-Server pro Projekt: PixelLab für Pixel Art, ElevenLabs für Audio, Meshy für 3D, Device-Test-Tools
• Kontextbewusster Chat, der je nach Projekttyp automatisch das passende Wissen lädt

FÜR WEN
Solo- und Kleinteam-Indie-Entwickler, die mehrere Spieleprojekte parallel betreiben und fernab des Schreibtischs produktiv bleiben wollen.
