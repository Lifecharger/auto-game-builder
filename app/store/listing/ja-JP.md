# Title
Auto Game Builder

# Short Description
スマホから開発PCを遠隔操作 — ビルド、AI自動化、タスク管理。

# Full Description
Auto Game Builderは、Auto Game Builderオーケストレーターのコンパニオンアプリです。あなたのスマホを、開発PC上のすべてのゲームプロジェクトのためのリモートコントロールパネルに変えます。

機能
• Flutter、Godot、Phaserプロジェクトのビルドをワンタップで起動
• AABをターミナルを開かずにGoogle Playへ直接アップロード
• Issue（バグ、クラッシュ、機能要望）を作成し、AIによる自動修正キューに追加
• プロジェクト毎のタスク管理、pending/in-progress/completed/failedでフィルタ
• AI自動化ループの起動・停止、間隔設定、セッションのリアルタイム監視

ドキュメント＆ナレッジ
• Game Design Document、AI指示書（CLAUDE.md）、Art Bibleの閲覧・編集
• AI支援のドキュメント改善でエージェントの指示を鋭く

可視性
• 全プロジェクト横断のログビューア
• ステータスバッジとタイムスタンプ付きのビルド履歴
• 進行中の自動化作業のライブセッションモニター

AIとMCP
• エンジン毎にClaude、Gemini、Codex、ローカルモデルを切替
• プロジェクト毎にMCPサーバーを有効/無効：ピクセルアート用PixelLab、オーディオ用ElevenLabs、3D用Meshy、デバイステストツール
• プロジェクトタイプに応じて関連知識を自動ロードするコンテキスト対応チャット

対象
複数のゲームプロジェクトを同時に管理し、デスクを離れても生産性を維持したいソロ／小規模インディー開発者。
