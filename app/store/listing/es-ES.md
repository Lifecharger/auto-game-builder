# Title
Auto Game Builder

# Short Description
Controla tu PC de desarrollo desde el móvil — builds, IA, tareas.

# Full Description
Auto Game Builder es la app compañera del orquestador Auto Game Builder. Convierte tu móvil en un panel de control remoto para cada proyecto de juego de tu PC de desarrollo.

QUÉ HACE
• Dispara builds de proyectos Flutter, Godot y Phaser con un toque
• Sube AABs directamente a Google Play sin abrir un terminal
• Crea issues (bugs, crashes, features) y encola auto-fixes impulsados por IA
• Gestiona tareas por proyecto con filtros por pending, in-progress, completed y failed
• Arranca y detiene bucles de automatización de IA, configura intervalos y ve sesiones en vivo

DOCS Y CONOCIMIENTO
• Consulta y edita Game Design Documents, instrucciones de IA (CLAUDE.md) y Art Bibles
• Mejora documental asistida por IA para afinar instrucciones de los agentes

VISIBILIDAD
• Visor de logs unificado entre proyectos
• Historial de builds con badges de estado y marcas de tiempo
• Monitor de sesión en vivo para trabajos de automatización en curso

IA Y MCP
• Alterna entre Claude, Gemini, Codex o modelos locales por motor
• Activa/desactiva servidores MCP por proyecto: PixelLab para pixel art, ElevenLabs para audio, Meshy para 3D, herramientas de testeo
• Chat contextual que carga el conocimiento apropiado según el tipo de proyecto

PARA QUIÉN ES
Desarrolladores indie solos y en equipos pequeños que llevan varios proyectos a la vez y quieren seguir siendo productivos lejos del escritorio.
