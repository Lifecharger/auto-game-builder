# Title
Auto Game Builder

# Short Description
Pilotez votre PC dev depuis votre mobile — builds, IA, tâches.

# Full Description
Auto Game Builder est l'app compagnon de l'orchestrateur Auto Game Builder. Elle transforme votre mobile en tableau de bord distant pour chaque projet de jeu sur votre PC de dev.

CE QU'ELLE FAIT
• Déclenchez des builds Flutter, Godot et Phaser d'un seul tap
• Envoyez des AAB directement sur Google Play sans ouvrir de terminal
• Créez des issues (bugs, crashes, features) et mettez-les en file pour auto-fix IA
• Gérez les tâches par projet avec filtres pending, in-progress, completed, failed
• Lancez et arrêtez les boucles d'automatisation IA, configurez les intervalles, suivez les sessions en direct

DOCS ET CONNAISSANCE
• Consultez et éditez Game Design Documents, instructions IA (CLAUDE.md), Art Bibles
• Amélioration documentaire assistée par IA pour affiner les instructions des agents

VISIBILITÉ
• Visualiseur de logs unifié tous projets
• Historique de builds avec badges d'état et timestamps
• Monitor de session en direct pour les automatisations en cours

IA ET MCP
• Bascule entre Claude, Gemini, Codex ou modèles locaux par moteur
• Active/désactive les serveurs MCP par projet : PixelLab pour le pixel art, ElevenLabs pour l'audio, Meshy pour la 3D, outils de test
• Chat contextuel qui charge les bonnes connaissances selon le type de projet

POUR QUI
Devs indés solos et petites équipes qui gèrent plusieurs projets de jeu simultanément et veulent rester productifs loin du bureau.
